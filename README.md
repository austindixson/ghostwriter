# ghostwriter

First draft Hermes skill submission for collaborative full-novel generation.

## Contents
- `SKILL.md` — Novel Forge Complete skill (v0.2.0)
- `method-plan.md` — extracted method and hackathon packaging plan

## Summary
This skill is designed to:
- intake a story idea via focused questions
- co-design characters/events/relationships/symbolism/foreshadowing
- generate organized planning artifacts and chapter drafts
- support iterative feedback and revision loops
- produce visual planning outputs (tables/charts/slide-style summaries)
- compile a final manuscript package after passing quality gates
